# 🎓 AI高考志愿填报助手

基于 Vue 3 + Vite 开发的智能志愿推荐系统，利用大语言模型为考生提供个性化志愿填报方案。

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## ✨ 功能特性

- 🧠 **AI智能推荐**：结合历年录取数据和就业趋势生成志愿方案
- 📊 **多维分析**：院校/专业/地域/分数四维评估体系
- 💬 **自然语言交互**：支持用对话方式优化推荐结果
- 📱 **响应式设计**：适配手机、平板和桌面设备
